Run using:
python ks_algorithms.py -inst <filename> -alg <method> -time <cutoff> -seed <seed>

assumes data located in DATASET\ so <filename> should be of form: "DATASET\small_scale\small_1"

Additional options:
<filename> = test, small, large --> runs all respective datasets

Must have output folder in directory with subfolders: test, small_scale, large_scale
